# Stock Watchlist Web App

## Overview
This project is a stock watchlist web application built using Vue. It allows users to search for stocks, view real-time stock data, and manage a personal watchlist.

## Tech Stack
- Vue
- Vue Router
- Pinia
- Finnhub API
- GitHub Pages (deployment)

## Features
- Search stocks by symbol or company name
- View current stock data (price, change, percent change)
- Add stocks to a watchlist
- Remove stocks from watchlist
- Watchlist stored using localStorage
